# 大学生就业与职业成长平台 - 需求规格说明书

## 1. 项目概述

### 1.1 项目背景
本平台旨在系统性解决大学生求职方向模糊、职业规划缺失、实习岗位获取渠道单一、简历制作不规范、缺乏面试实战经验等核心痛点问题，同时帮助企业精准匹配适配的应届毕业生与实习生，构建高效的人才供需生态系统。

### 1.2 目标用户
| 用户角色 | 描述 | 核心需求 |
| :--- | :--- | :--- |
| 高校学生 | 在校大学生、应届毕业生 | 岗位搜索、简历优化、职业测评、面试练习、职业规划 |
| 企业HR | 企业人力资源管理人员 | 岗位发布、简历筛选、在线面试、人才匹配 |
| 平台管理员 | 平台运营管理人员 | 用户管理、内容管理、系统配置、数据统计 |

### 1.3 平台目标
- 打通学生求职、企业招人、职业能力培养全流程
- 首月注册学生用户500+，企业用户50+
- 降低企业招聘成本30%以上

## 5. 数据流向

### 5.1 岗位申请流程
```
学生搜索岗位 → 查看岗位详情 → 投递简历 → HR查看申请 → 筛选简历 → 发送面试邀请 → 在线面试 → 发放offer
```

### 5.2 职业测评流程
```
学生选择测评 → 完成测评题目 → 系统分析 → 生成测评报告 → 提供职业建议 → 更新职业规划
```

### 5.3 简历优化流程
```
学生创建/上传简历 → AI分析简历 → 生成优化建议 → 用户修改简历 → 重新评分
```
